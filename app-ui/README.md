# OSM Finder - Frontend

Frontend React/TypeScript para la aplicación OSM Finder.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`.

## Build para Producción

```bash
npm run build
```

Los archivos compilados estarán en `dist/`.

## Estructura

- `src/App.tsx`: Componente principal
- `src/components/MapView.tsx`: Componente del mapa con Leaflet
- `src/components/SearchPanel.tsx`: Panel de búsqueda
- `src/components/ResultsList.tsx`: Lista de resultados
- `src/types.ts`: Definiciones de tipos TypeScript

## Tecnologías

- **React 18**
- **TypeScript**
- **Vite** (build tool)
- **Leaflet** (mapas)
- **OpenAI Apps SDK** (integración con ChatGPT)
